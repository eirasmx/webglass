API

CSS-Driven

Drop a single CSS custom property on any element. Children inherit glass settings from their parent via the cascade.

.card {
  --wg-render: true;
  --wg-shape: squircle;
  --wg-refraction: 1.6;
  --wg-scale: 0.22;
  --wg-blur: 0.005;
}
